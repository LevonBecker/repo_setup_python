# HowTo: Setup Local Workstation
These are the steps to set up your dev workstation or CI worker to use the Python Invoke CLI Wrapper.

## Assumptions
* macOS (BASH Shell)

## Prerequisites
* [uv](https://docs.astral.sh/uv/)
    * macOS
        * `brew install uv`

#### Install Python Virtual Environment and Project Libraries
`./setup.sh`
